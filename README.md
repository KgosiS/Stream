# Go-Shop

Go-Shop is a dynamic Android marketplace app that allows users to buy and sell products locally. Users can browse the latest items, contact sellers directly via WhatsApp or email, and manage their favorite products. The app is powered by **Firebase** for authentication, Firestore for product data, and Realtime Database for profile images.

---

## Features

- **User Authentication:** Secure sign-in and sign-up using Firebase Authentication.
- **Product Browsing:** View latest products and filter by brand.
- **Favorites:** Add products to favorites and sync across devices.
- **Product Details:** View detailed information including images, description, price, and ratings.
- **Contact Sellers:** Contact sellers via WhatsApp or email directly from the app.
- **Profile Management:** View and edit your profile, upload a profile picture.
- **Upload Items:** Users can list their own products for sale.
- **Mobile-Responsive UI:** Designed with modern Android UI/UX patterns.

---



---

## Tech Stack

- **Android:** Kotlin, Android Jetpack components
- **Database:** Firebase Firestore & Realtime Database
- **Authentication:** Firebase Authentication
- **Image Loading:** Glide
- **Local Storage:** Room Database for favorites
- **Coroutines:** For asynchronous tasks
